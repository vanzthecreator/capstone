# Life Signal Backend

This is the Express.js backend for the Life Signal application.

## Purpose
This server is designed to handle **heavy computational tasks** that would otherwise drain the mobile device's battery or cause performance issues on the frontend.

## Key Responsibilities
1.  **Media Compression**: Processing high-resolution images and videos before storage.
2.  **Advanced Search**: Executing complex search algorithms across the user database.
3.  **Notifications**: Managing push notification logic.

## How to Run
1.  Navigate to this folder: `cd backend`
2.  Install dependencies: `npm install`
3.  Start the server: `npm start`
